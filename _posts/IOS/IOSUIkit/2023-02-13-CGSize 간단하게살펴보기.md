# CGSize 간단하게살펴보기
- 너비와 높이값을 나타내는 구조체
~~~
public struct CGPoint {
  public var width: CGFloat
  public var height: CGFloat
  public init()
  public init(width: CGFloat, height: CGFloat)
}
~~~

내일 부턴 SwiftUI에 대해 집중적으로 공부해 봐야겠다.
