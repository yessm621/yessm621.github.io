## 6. Modal

modal.html

```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>CSS Flex</title>
    <link rel="stylesheet" href="default.css">
    <link rel="stylesheet" href="ui.css">
</head>
<body>
    <div class="modal">
        <div class="dialog">
            Lorem ipsum dolor sit amet, consectetur adipisicing elit. Asperiores, et. Dolor facilis architecto sunt, quis eius animi ea vel labore? Placeat, error optio amet expedita qui provident consequatur blanditiis nihil, atque, accusantium aliquid nisi eaque pariatur ratione inventore. Rerum, totam.
        </div>
    </div>
</body>
</html>
```

```css
/* modal */
.modal {
	/* flex 설정 주고 가로축, 세로축 center */
    display: flex;
    justify-content: center;
    align-items: center;
	/* 배경화면을 고정 */
    position: fixed;
	/* 상하좌우 여백 0 */
    left: 0;
    right: 0;
    top: 0;
    bottom: 0;
	/* 투명도까지 설정 */
    background: rgba(0,0,0,0.3);
}

.dialog {
    width: 50vw;
    padding: 2em;
    border-radius: 1em;
    background: white;
}
```

```css
.card-item {
    display: flex;
    /* flex 의 메인축을 세로방향으로 설정 */
    flex-direction: column;
    margin-bottom: 2rem;
}

.card-image {
    height: 0;
    padding-bottom: 60%;
    background-color: lightgray;
    background-repeat: no-repeat;
    background-position: center;
    background-size: cover;
}

.card-image img {
    display: none;
}

.card-desc {
    /* description 의 길이가 다르기 때문에 
    높이를 동일하게 맞춰주려면 flex-grow 를 사용 */
    flex: 1 1 auto;
    padding: 1em;
    background: white;
}

@media (min-width: 600px) {
    .card-list-con {
        /* 세로축 스크롤바만 활성화 */
		overflow-x: hidden;
	}

    .card-list {
        display: flex;
        /* flex-wrap: wrap; 크기를 벗어나면 아래행으로 이동 */
        flex-wrap: wrap;
        margin: 0 -1rem;
    }

    .card-item {
        width: 50%;
        padding: 0 1rem;
    }
}

@media (min-width: 1200px) {
    .card-item {
        width: 33.33333%;
    }
}
```
